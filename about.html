Sobre o Projeto

Este site foi desenvolvido como parte de um exercício prático, com o objetivo de aplicar conhecimentos de HTML, CSS, JavaScript e outras tecnologias web. A página principal inclui links de navegação para diversas seções, e o formulário HTML contém no mínimo 5 campos interativos. A integração de uma folha de estilos em CSS proporciona um layout personalizado e agradável.
